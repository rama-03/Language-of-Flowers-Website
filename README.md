This project is a simple webpage about six flowers and their symbolic meanings in the language of flowers. It was created as part of the Web and Applications Development track to practise the basics of HTML and CSS.

The webpage presents the flowers in a two-column layout. Each flower section includes a square image, the flower's name, a short description, and its meaning in flower language. The page also uses background images, centered content, and custom styling to create a soft floral theme.

## Flowers Included

- Fuchsia
- Orchid
- Peonies
- Canterbury Bells
- Dicentra
- Wisteria

- Six flower sections arranged in two columns

- Custom page and title background images
- Times New Roman font
- Centered layout and content

## How to View the Website

1. Download or clone this repository.
2. Make sure the HTML file and the "Images" folder remain in the same project folder.
3. Open the HTML file in a web browser.
