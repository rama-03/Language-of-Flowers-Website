This project is a simple webpage about six flowers and their symbolic meanings in the language of flowers.



The webpage presents the flowers in a two-column layout. Each flower section includes a square image, the flower's name, a short description, and its meaning in flower language. The page also uses background images for the title, centered content, and custom styling to create a soft floral theme.



\## Flowers Included



\- Fuchsia

\- Orchid

\- Peonies

\- Canterbury Bells

\- Dicentra

\- Wisteria

1\. Download or clone this repository.
